## For Developers

Feel free to use this portfolio template for your own personal website! It's open source and available for everyone.
- You can clone the repo and customize the `lib/constants/app_constants.dart` file with your own data.
- The design is responsive and ready to deploy.

**Note on Firebase:**
This project uses **Firebase** for:
- **Hosting**: To serve the web app.
- **Analytics**: To track page views and interactions.

If you fork this repo, you will need to:
1.  Create your own Firebase Project.
2.  Install the FlutterFire CLI: `npm install -g firebase-tools` and `dart pub global activate flutterfire_cli`.
3.  Run `flutterfire configure` to generate your own `firebase_options.dart`.
4.  Update `firebase.json` and `.firebaserc` with your project ID.
